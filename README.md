# Rockwell_Modbus_TCP_Program

This Modbus Example communicates AB Compactlogix PLC to Applied Motion Products MDX Motor using Modbus TCP
This program was developed in Studio5000 v32

Applied Motion Products MDX Integrated Servo Motor:- **https://www.applied-motion.com/s/product/mdxt62g-ip000/01tJ30000033rHMIAY?name=MDXT62GNBIPA000-400W-IP65-MDX-w-17-bit-Absolute-Encoder**
Modbus Register mapping manual:- **https://applied-motion.s3.amazonaws.com/documents/Manuals/User%20Manual_Modbus_RevB_920-0176.pdf**
